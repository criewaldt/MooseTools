Before you begin

Important Reminders

Remove any staff banks from the cash bag before using this tool.

Always double-check (or even triple-check) your cash counts.

If there is a 1% support bonus due, this tool will not tell you how to divide it among the support staff. That is still the responsibility of the closing server.



Instructions

Run checkouts on both terminals to get the checkout reciept for each.

Remove any banks brought by staff from the cash bag, but do not remove any other cash from the bag before using this tool, including the deposit.

Complete the server checkout form using the numbers from the top of each terminal checkout reciept, and the cash bag count. Include 0's in the form fields if the value for an item is '0'.

Once you've created your checkout report using this tool, you will have all the numbers you need to make each envelope needed for the shift deposit & employee tipouts.
